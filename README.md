# Picasso Integration Tester

Picasso Integration Tester is a collection of different implementation tests for the Picasso Polkadot Parachain.

## Installation


```bash
$ npm ci
```

## Usage

### To run the devnet dummy data initializer
**(Not yet implemented!)**
```bash
$ npm run init
```

### To run the tests:
```bash
$ npm run test
```
